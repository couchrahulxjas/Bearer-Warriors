# Warriors - Mental Health Platform

A React + TypeScript + Vite frontend with Node.js/Express backend using dummy psychiatrist data for mental health support and psychiatrist booking.

## 🚀 Quick Start

### Frontend
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Backend (Dummy Data - No Database Required)
```bash
cd backend

# Install dependencies
npm install

# Start backend server
npm run dev
```

**Available API**: `http://localhost:5000`

## 📊 Backend Features

- ✅ No database setup required - uses in-memory dummy data
- ✅ 20 pre-loaded psychiatrist records with complete profiles
- ✅ REST API endpoints for psychiatrist data
- ✅ Real-time data - add/modify psychiatrists on the fly
- ✅ Instant startup - no connection timeouts
